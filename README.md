# ことば10

中国語・韓国語・エスペラント語・フィンランド語を、1セット10問の4択形式で学習するスマートフォン対応Webアプリです。

**現在のバージョン：v1.4.0**  
**リリース日：2026年7月19日**

## 収録語彙

| 言語 | 語彙数 | おおよその難易度 |
|---|---:|---|
| 中国語（簡体字・繁体字） | 260語 | A2〜B1程度 |
| 韓国語 | 110語 | A1〜A2程度 |
| エスペラント語 | 260語 | A2〜B1程度 |
| フィンランド語 | 150語 | A2程度 |

難易度はCEFRの学習場面を参考にした目安です。公式の認定語彙表ではありません。

## 主な機能

- 日本語→外国語、外国語→日本語、または1問ごとのランダム出題
- 1セット10問の4択問題
- 中国語（簡体字・繁体字）、韓国語、エスペラント語、フィンランド語に対応
- 正解数、正答率、学習セット数、回答数の記録
- 間違えた単語の一覧表示
- 間違えた単語だけを出題する復習モード
- 単語ごとの正解回数・誤答回数の記録
- 例文と日本語訳の表示
- 正解時・不正解時の追加学習リンク
  - 単語をGoogleで検索
  - 例文をGoogle翻訳で日本語に翻訳
  - 例文をChatGPTで文法的に解説
- 誤答一覧から単語をタップし、同じ3種類の追加学習リンクを利用可能
- 中国語の誤答一覧にピンインを表示
- 学習履歴と各問題の回答内容の保存
- JSON形式での学習記録のバックアップ・復元
- PWAとしてホーム画面へ追加可能
- Service Workerによるオフライン利用

## ChatGPT例文解説

「例文をChatGPTで解説」を押すと、新しいタブでChatGPTを開き、表示中の例文を次の形式で渡します。

```text
次の〇〇語の文章を文法的に解説してください。

［例文］
```

例：

```text
次のフィンランド語の文章を文法的に解説してください。

Etsimme uutta asuntoa läheltä keskustaa.
```

## 誤答一覧

誤答一覧では、単語をタップすると詳細が開きます。

- 例文
- 例文の日本語訳
- Google検索
- Google翻訳
- ChatGPTによる文法解説

中国語では、単語と日本語訳に加えてピンインも一覧に表示します。

## 復習の仕組み

一度でも間違えた単語は「要復習」として記録されます。復習モードで2回連続正解すると「要復習」が解除されます。再度間違えると、もう一度「要復習」に戻ります。

## GitHubへの更新方法

この更新パッケージ内の次のファイルを、GitHubリポジトリ最上位にある同名ファイルと差し替えてください。

```text
app.js
words.js
index.html
manifest.json
service-worker.js
README.md
```

変更をコミットしてから、GitHub Pagesへの反映を数分待ちます。

スマートフォンに旧版が残る場合は、次の順で操作してください。

1. 「ことば10」を完全に終了する
2. Safariで公開URLを直接開く
3. ページを再読み込みする
4. ホーム画面版を再起動する
5. 改善しない場合はホーム画面アイコンを削除し、再度追加する

## 学習記録の保存

学習記録はブラウザーの`localStorage`に保存されます。今回の更新で既存の履歴、誤答記録、設定は削除されません。

設定画面から定期的にJSONバックアップを書き出してください。

## 外部サービス

Google検索、Google翻訳、ChatGPTによる例文解説にはインターネット接続が必要です。外部ページは新しいタブまたはウィンドウで開き、元の「ことば10」の画面は維持されます。

## フィンランド語データ形式

```javascript
{
  "id": "fi001",
  "word": "asunto",
  "japanese": "住居・アパート",
  "example": "Etsimme uutta asuntoa läheltä keskustaa.",
  "exampleJapanese": "私たちは中心街の近くで新しい住居を探しています。",
  "category": "住居"
}
```
