# ことば10

中国語・韓国語・エスペラント語・フィンランド語を、1セット10問の4択形式で学習するスマートフォン対応Webアプリです。

**現在のバージョン：v1.5.1**  
**リリース日：2026年7月20日**

## 収録語彙

| 言語 | 語彙数 | おおよその難易度 |
|---|---:|---|
| 中国語（簡体字・繁体字） | 260語 | A2〜B1程度 |
| 韓国語 | 110語 | A1〜A2程度 |
| エスペラント語 | 260語 | A2〜B1程度 |
| フィンランド語 | 150語 | A2程度 |

## 主な機能

- 日本語→外国語、外国語→日本語、またはランダム出題
- 1セット10問の4択問題
- 学習履歴、正答率、単語別成績の保存
- 復習リストと、復習リストからの10問練習
- 間違えた単語を復習リストへ自動追加
- 正解した単語も回答後に任意で復習リストへ追加
- 復習リストの単語をタップして、和訳、例文、追加学習メニューを表示
- 中国語の復習リストにピンインを表示
- 復習リストの例文暗記モード
  - 日本語訳だけを一覧表示
  - 日本語訳をタップすると外国語の例文を表示
- Google検索、Google翻訳、ChatGPTによる例文解説
- JSON形式のバックアップと復元
- PWA、スマートフォン、オフライン利用に対応

## 復習リスト

問題を間違えると、その単語は自動的に復習リストへ追加されます。正解した場合でも、回答後の「復習リストに追加」ボタンから任意に追加できます。

復習リストの一覧では、外国語の単語だけを表示し、和訳は隠します。単語をタップすると次の内容を表示します。

- 単語の和訳
- 外国語の例文と日本語訳
- この単語をGoogleで検索
- 例文をGoogle翻訳で確認
- 例文をChatGPTで解説

復習モードで2回連続正解すると「要復習」から「復習済み」へ変わりますが、単語は復習リストに残るため、後から再確認できます。

## 例文暗記モード

復習画面の「例文暗記」を選ぶと、選択した言語について、復習リスト内の例文の日本語訳だけを一覧表示します。

日本語訳をタップすると、対応する外国語の例文が開きます。日本語から外国語の文章を思い出す練習に利用できます。

## ChatGPT例文解説

「例文をChatGPTで解説」を押すと、新しいタブでChatGPTを開き、次の形式で例文を渡します。

```text
次の〇〇語の文章を文法的に解説してください。

［例文］
```

## 更新方法

この更新パッケージ内の次のファイルを、GitHubリポジトリ最上位にある同名ファイルと差し替えてください。

```text
app.js
index.html
service-worker.js
README.md
```

`words.js`、`manifest.json`、`style.css`の差し替えは不要です。

GitHub Pagesへの反映を数分待ち、スマートフォンで旧版が残る場合は、アプリを完全に終了してSafariから公開URLを再読み込みしてください。

## 学習記録

学習記録はブラウザーの`localStorage`に保存されます。v1.5.0では、従来の誤答記録を自動的に復習リストへ引き継ぎます。既存の履歴や成績は削除されません。
