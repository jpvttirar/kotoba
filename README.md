# ことば10

中国語・韓国語・エスペラント語を、1セット10問の4択形式で学習するスマートフォン対応Webアプリです。

**現在のバージョン：v1.3.0**  
**リリース日：2026年7月19日**

## 収録語彙

| 言語 | 語彙数 | おおよその難易度 |
|---|---:|---|
| 中国語（簡体字・繁体字） | 260語 | A2〜B1程度 |
| 韓国語 | 110語 | A1〜A2程度 |
| エスペラント語 | 260語 | A2〜B1程度 |

中国語は同じ語彙データに簡体字・繁体字・ピンインを収録し、学習時に表記を切り替えます。

## 主な機能

- 日本語から外国語、外国語から日本語、または1問ごとのランダム出題
- 1セット10問の4択問題
- 中国語（簡体字・繁体字）、韓国語、エスペラント語に対応
- 正解数、正答率、学習セット数、回答数の記録
- 間違えた単語の一覧表示
- 間違えた単語だけを出題する復習モード
- 単語ごとの正解回数・誤答回数の記録
- 例文と日本語訳の表示
- 正解時・不正解時の追加学習リンク
  - 単語をGoogleで検索
  - 例文をGoogle翻訳で日本語に翻訳
  - 例文をChatGPTで文法的に解説
- 学習履歴と各問題の回答内容の保存
- JSON形式での学習記録のバックアップ・復元
- スマートフォン対応
- PWAとしてホーム画面へ追加可能
- Service Workerによるオフライン利用

## 復習の仕組み

一度でも間違えた単語は「要復習」として記録されます。

復習モードで正解を重ねると、苦手語から外れます。現在の設定では、復習モードで2回連続正解すると「要復習」が解除されます。再度間違えた場合は、もう一度「要復習」に戻ります。

kotoba/

### 各ファイルの役割

- `index.html`：画面の入口とファイルの読み込み
- `style.css`：スマートフォン向けデザイン
- `app.js`：出題、採点、履歴、復習、バックアップなどの処理
- `words.js`：単語、読み方、意味、例文のデータ
- `manifest.json`：ホーム画面追加時のアプリ情報
- `service-worker.js`：キャッシュ管理とオフライン対応
- `icon-192.png`、`icon-512.png`：アプリアイコン
- `README.md`：この説明書

## GitHub Pagesで公開する方法

1. GitHubにログインします。
2. `kotoba`リポジトリを開きます。
3. アプリのファイルをリポジトリの最上位に置きます。
4. `Settings` → `Pages` を開きます。
5. `Source`を`Deploy from a branch`にします。
6. Branchを`main`、フォルダーを`/(root)`にします。
7. 保存し、数分後に公開URLを開きます。

公開URLの例：

```text
https://ユーザー名.github.io/kotoba/
```

`index.html`は、`vocab-app`などの下位フォルダーではなく、リポジトリの最上位に置いてください。

## アップデート方法

更新版で指定されたファイルを、GitHub上の同名ファイルと置き換えてコミットします。

スマートフォンで古い画面が残る場合は、次の順で確認してください。

1. GitHub Pagesへの反映を数分待つ
2. アプリを完全に終了する
3. Safariで公開URLを直接開いて再読み込みする
4. ホーム画面版を再起動する
5. 改善しない場合はホーム画面のアイコンを削除し、再度追加する

`index.html`の`app.js`読み込みURLと、`service-worker.js`のキャッシュ名は、更新時に変更して古いキャッシュが残りにくいようにしています。

## 学習記録の保存

学習記録はブラウザーの`localStorage`に保存されます。

次の場合には記録が失われることがあります。

- ブラウザーのWebサイトデータを削除した
- 別のブラウザーを使用した
- 端末を変更した
- GitHub Pagesの公開URLを変更した

設定画面から定期的にバックアップを書き出してください。バックアップには、アプリのバージョン、データ形式のバージョン、設定、単語別成績、学習履歴が含まれます。

## 外部サービスを使う機能

Google検索、Google翻訳、ChatGPTによる例文解説にはインターネット接続が必要です。リンクは新しいタブまたはウィンドウで開き、元の「ことば10」の画面は維持されます。

ChatGPTのリンクでは、表示中の例文と参考訳を使い、次の内容を解説するプロンプトを渡します。

- 単語ごとの意味
- 文法構造
- 自然な日本語訳
- 初学者向けの説明

## 単語を追加する方法

`words.js`の該当する言語配列に、既存データと同じ形式で追加します。IDは重複しないようにしてください。

### 中国語

```javascript
{
  "id": "zh261",
  "simplified": "旅行",
  "traditional": "旅行",
  "pinyin": "lǚxíng",
  "japanese": "旅行",
  "exampleSimplified": "我喜欢旅行。",
  "exampleTraditional": "我喜歡旅行。",
  "exampleJapanese": "私は旅行が好きです。",
  "category": "日常"
}
```

### 韓国語

```javascript
{
  "id": "ko111",
  "word": "여행",
  "japanese": "旅行",
  "example": "저는 여행을 좋아해요.",
  "exampleJapanese": "私は旅行が好きです。",
  "category": "日常"
}
```

### エスペラント語

```javascript
{
  "id": "eo261",
  "word": "vojaĝo",
  "japanese": "旅行",
  "example": "Mi ŝatas vojaĝi.",
  "exampleJapanese": "私は旅行することが好きです。",
  "category": "日常"
}
```
